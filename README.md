# Проект "фильтр фотографий"

## Описание проекта
Данный проект создан для помощи выбора фильтра для фотографий. Решит проблему с фильтрами.

## Цели проекта
- Определение с  фильтрами
- применение филтров к фотографиям
## Используемые технологии
- Python для создание выбора фильтров
## Устоновка зависимостей
Загрузите на компьютер необходимые файлы, затем используйте \`pip\` для установки зависимостей и установить зависимости. Зависимости можно установить командой, представленной ниже.Установите зависимости командой:

```
pip install -r requirements.txt
```

## Пример запуска
Для запуска скрипта у вас уже должен быть установлен Python3.
Для получения списка фотографий, необходимо записать:
```
python main.py --stray_animals
```
Для получения основного финансового фонда программы, необходимо записать:
```
python main.py --financial_fund
```

Проект выполнен в образовательных целях на онлайн-курсе "Основы Python" школы "Лидер".
